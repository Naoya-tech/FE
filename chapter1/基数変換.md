## 10進数から2進数の変換

### ex. 13
```
13 / 2 = 6 ... 1  
6  / 2 = 3 ... 0  
3  / 2 = 1 ... 1  
1  / 2 = 0 ... 1  

A. 1101
```


### ex. 13.375
```
・整数部分
  上記の通り 1101
.小数部分
  0.375 * 2 = 0. 75
  0.75  * 2 = 1. 5
  0.5   * 2 = 1. 0

  よって小数部分は .011

A. 1101.011
```


## 2進数から8進数16進数の変換

### ex. 1101.01
```
3桁ごとに区切ると
001 /101 . 010

001 部分       101部分        010部分
  (0 * 4)       (1 * 4)       (0 * 4)
+ (0 * 2)      +(0 * 2)       +(1 * 2)
+ (1 * 1)      +(1 * 1)       +(0 * 1)
= 1            = 5            = 2

よって
1101.01(2進数) ＝ 15.2(8進数)
```
